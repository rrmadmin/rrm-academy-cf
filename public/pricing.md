# RRM Academy Pricing

RRM Academy is a project of the Restorative Reproductive Medicine Foundation, a 501(c)(3) nonprofit organization (EIN 93-4594315). All research library content, pillar guides, commentary, FAQs, and the glossary are freely accessible without an account. Courses are one-time purchases or free, depending on the course.

- Site: https://rrmacademy.org
- Course catalog: https://rrmacademy.org/courses/
- Research library (free): https://rrmacademy.org/library/
- Contact: info@rrmacademy.org

## API access

RRM Academy exposes its research library to AI agents via a Model Context Protocol (MCP) server. Access requires a Bearer API key. Contact info@rrmacademy.org to request access for production agent integrations. There is no self-serve billing tier.

- Agent API spec: https://rrmacademy.org/openapi.json
- MCP discovery: https://rrmacademy.org/.well-known/mcp.json

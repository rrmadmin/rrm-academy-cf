# Endo Survey: ICD-10-CM Symptom Mapping (INTERNAL USE ONLY)

> **NOT for patient-facing materials.** This maps patient-reported symptoms to ICD-10-CM codes. Pairing patient complaints with diagnostic codes constitutes practicing medicine. Use for research, grants, internal clinical reference only.

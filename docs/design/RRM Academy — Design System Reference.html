Accessibility: Purple 700 (#725e7e) is the minimum for text on white — passes AA at 5.8:1. Purple 500 (#987da8) fails for body text — use only on large headings (24px+). Purple 900 (#4c3e54) passes AAA at 9.9:1 — use for maximum contrast.

Usage Rules

Do
  • Use Purple 700 (#725e7e) for primary buttons, links, and nav
  • Use Purple 900 (#4c3e54) for hover states
  • Use Neutral 900 (#313131) for body text
  • Alternate section backgrounds: white, neutral-100, purple-50, cream
  • Keep heading hierarchy: Display > H1 > H2 > H3 > H4
Don't
  • Don't use Purple 500 (#987da8) for body-size text — fails accessibility
  • Don't use the same color for button default and hover
  • Don't use more than 2 heading sizes on one section
  • Don't introduce new colors outside this palette
  • Don't mix section background patterns — pick one flow per page
